 
En el presente trabajo se realizó una página web para la Entrega del TP final turno noche lun-mie. Se utilizó visual studio code como editor de texto incorporando diversas tecnologías como MySQL como base de dato (se eligió este sistema de gestión de datos dado que MongoDB se suele utilizar para proyectos más extensos) al no tratarse de una red social, por nombrar un ejemplo se optó por llevar a cabo el proyecto en MySQL almacenando los datos en tablas y con una sintaxis un poco menos flexible sin afectar rendimiento.
Como entorno de trabajo el uso de express, morgan, así como también nodemon para actualizar el servidor en tiempo real sin recargar la página con Node.js
Una web básica utilizando CRUD para registrarse y guardar sus datos.
